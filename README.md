
# Snake Game

Desktop version of famous snake game created in python with tkinter library.

## Run Locally

Clone the project

```bash
  git clone https://github.com/johnnyweak1/Python-SnakeGame
```

Go to the project directory

```bash
  cd Python-SnakeGame
```

Run script

```bash
  python3 main.py
```
